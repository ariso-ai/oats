# Getting help with oats

Glad you're here! Here's the fastest way to get what you need.

| I want to… | Go here |
| --- | --- |
| 🐛 **Report a bug** | [Open a bug report](https://github.com/ariso-ai/oats/issues/new?template=bug_report.yml) |
| 💡 **Request a feature** | [Open a feature request](https://github.com/ariso-ai/oats/issues/new?template=feature_request.yml) |
| 💬 **Ask a question / share feedback** | [Discussions](https://github.com/ariso-ai/oats/discussions) |
| 🔐 **Report a security issue** | See [SECURITY.md](SECURITY.md) — please don't open a public issue |

| 🔧 **Set up a dev environment / contribute** | [CONTRIBUTING.md](CONTRIBUTING.md) |

## Before you post

- Make sure you're on the **latest release** — oats auto-updates, and many issues are already fixed. Check the [Releases page](https://github.com/ariso-ai/oats/releases/latest).
- Skim existing [issues](https://github.com/ariso-ai/oats/issues) and [discussions](https://github.com/ariso-ai/oats/discussions) — your question may already be answered.
- When reporting a problem, include your **oats version**, **macOS version**, and which **transcription backend** (Ariso cloud or Local) you're using. The bug report template prompts you for all of this.

We're a small team — thanks for your patience, and thanks for using oats.
